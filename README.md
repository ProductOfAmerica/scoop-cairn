# scoop-cairn

Scoop bucket for [cairn](https://github.com/ProductOfAmerica/cairn) — a verification substrate for AI coding agents.

## Install

```powershell
scoop bucket add cairn https://github.com/ProductOfAmerica/scoop-cairn
scoop install cairn
```

## Update

```powershell
scoop update cairn
```

Manifest `autoupdate` + `checkver` track GitHub releases; bumps are mechanical.
